InBiz

Web application which gives the users information about the world of local businesses

In-Biz is a web application that allows interested users to easily get infomation about the world of local businesses and observe interesting correlation between, for example, the location and the ratings and reviews of a local business. The data used for the analyses present on the website has been provided by Yelp.com

This was a university project, for the 2013's Social Data Modelling class at DTU. The project was completed in a tem of 4, together with Ruxandra Nistor, Cosmin Stefan-Dobrin and Rafaela Voiculescu.

The website show statistics and information gathered from the local businesses posted on yelp.com from 78 European cities. The data processing was done in python and the server part was based on Google AppEngine. The charts and graphs were done using the Google Charts API. The project can be found at in-biz.appspot.com.
