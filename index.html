Hello, I'm Marius

Welcome to my corner of the Internet

I'm an engineer in Computer Science living in Copenhagen, Denmark. I'm finishing my master studies and I'm working as an open source iOS developer on this project

I'm always working on something, either a personal project, a university assignment, an open source project or simple experiments with new technologies. If I'm not at my computer, I'm probably outdoors. Besides computers, technology and mobile, I like sports, board games and travelling to new places.

Sometimes I write here about iOS, Cocoa and other cool technologies I experiment with. I'm trying to do this at least once a month, but I'm not sure how this will work out. The blog also has a GitHub repo with its configuration and with open issues for future articles.

I'm not doing only iOS programming (at least, for now, as a student :) ). If you want, you can check out my GitHub profile. Also, some of my projects are described in My work section.

One way to learn and find out about the latest updates in your domain is to follow what some of the better guys are doing. I've noticed that a lot of the top iOS developers are very active on Twitter. So that's a great way to learn new things. All you have to do is follow the right guys.
