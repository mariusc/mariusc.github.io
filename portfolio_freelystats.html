FreelyStats

WebApp showing statistics and information about the freelancing market

FreelyStats is a web-application that provides a simple and interactive interface allowing the users to get relevant information according to their interests, regarding the projects, skills, freelancers, bids and others on the freelancing market.

This was a university project, for the 2012's Web and Social Interaction class at DTU. The project was completed in a tem of 3, together with Cosmin Stefan-Dobrin and Ruandra Nistor.

The website show statistics and information gathered from the projects posted on freelancer.com between January 2008 and November 2012. The data processing was done in python and the server part was based on Google AppEngine. The charts and graphs were done using the Google Charts API. The project can be found at freelystats.appspot.com.
